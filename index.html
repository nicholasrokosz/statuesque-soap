Tasks

You will be editing the following files: index.html, css/main.css and js/app.js.

  2. Add .completed onto each list item you have completed (and everyone here on after).

  4. Create a form in the div.container, then create a text input in the form.
  5. On the input, add a place holder of What needs to be done?
  6. Make the #tasks ol li elements turn the cursor to a pointer
  7. Capture the submit event on the form and prevent it from reloading the page.
  8. On submit, get the input value and append it to the #tasks ol
  9. Using CSS Only: If elements with the class .name on them is not empty, add a comma at the end of the element. (Bonus Points)
